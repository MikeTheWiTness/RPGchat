# RPGchat — AI 驱动的 TRPG 跑团系统

一个本地运行的 TRPG（桌上角色扮演游戏）模拟引擎。你提供战役背景，AI 接管 GM 角色：为每个 NPC 维护独立的角色档案和上下文，管理环境信息的多角色视角，在角色与环境之间智能调度发言顺序。

支持两种模式：**玩家参与模式**（你控制 PC 参与故事）和 **玩家空缺模式**（全自动推演，AI 小说生成器）。

## 快速开始

### 安装

```bash
git clone https://github.com/MikeTheWiTness/RPGchat.git
cd RPGchat
pip install -e .
```

### 配置 API Key

```bash
export OPENAI_API_KEY="sk-..."
```

### 运行

```bash
# 玩家参与模式 — 从模组启动
python cli.py --module "铁炉镇冒险"

# 玩家空缺模式（自动推演）— 从模组启动
python cli.py --module "新东京暗流" --auto

# 自由格式开始
python cli.py
```

### 基本玩法

在游戏中，你通过自然文本 + 标点标记来控制角色行为：

| 标记 | 含义 | 示例 |
|------|------|------|
| `【...】` 或 `"..."` | 对话（角色说的话） | `【小心身后！】` |
| `（...）` | 内心活动 | `（这个人看起来很可疑）` |
| 裸文本 | 行动/动作 | `我拔剑砍向哥布林` |
| `{...}` | 给 AI 的元指令 | `{继续}` / `{到我}` |

### 常用指令

| 指令 | 作用 |
|------|------|
| `{继续}` | 采纳 AI 推荐的发言者，推进一步 |
| `{到我}` | AI 连续推进直到需要你发言 |
| `{调查 描述}` | 定向查询某个动作的结果（如看信内容） |
| `{在场}` | 查看当前场景在场角色 |
| `{回滚 N}` | 回退到指定快照点重新推演 |
| `{历史}` | 查看最近快照列表 |
| `{存档}` / `{读档}` | 保存/加载游戏进度 |
| `{检查点}` | 触发上下文压缩 |

## 功能特性

- 🎭 **独立角色上下文**：每个 NPC 拥有独立的性格档案和动作历史
- 🧠 **Narrative 统一输出**：NPC 以自然叙述方式输出，非机械的三字段格式
- 📍 **场景追踪**：自动维护在场人员列表，处理角色入场/离场
- ⚖️ **信息不对称**：悄悄话和隐藏线索仅对目标角色可见
- 🎲 **技能检定**：可选的 d20 骰子系统（通过 `{检定 技能名}` 触发）
- 🍀 **运势系统**：每个 NPC 每阶段拥有运势等级影响行为倾向
- 💾 **存档加载**：本地持久化，中断可继续
- 🔙 **快照回滚**：任意回退到历史点重新推演
- 🤖 **玩家空缺模式**：全自动推演，AI 小说生成器

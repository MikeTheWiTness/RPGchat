# Issue #6: 上下文组装器 (ContextAssembler)

## What to build

实现为 LLM 调用组装完整 prompt 上下文的模块。

- `assemble_for_character(character_id)` — 为 NPC 生成动作单元组装上下文：
  - 该 NPC 的角色档案
  - 该 NPC 的角色上下文（动作单元历史，含内心活动）
  - 完整的公开对话历史
  - 环境上下文中对该 NPC 可见的条目
  - 当前在场人员名单
  - **不允许**包含其他角色的内心活动
- `assemble_for_judgment()` — 为判断机制组装全知上下文：
  - 所有角色的完整上下文（**含**内心活动）
  - 完整的环境上下文（全部条目）
  - 完整的公开对话历史
  - 当前在场人员名单
- 返回结构化的 prompt 友好格式（文本拼接），具体 prompt 模板由 LLMGateway 定义，此模块只负责数据聚合

## Acceptance criteria

- [ ] 为 NPC 组装时不泄露其他角色的内心活动
- [ ] 为判断机制组装时包含所有角色的内心活动
- [ ] 环境可见性过滤正确（只返回 visible_to 含该角色的条目）
- [ ] 当前在场人员名单正确包含
- [ ] 单元测试验证组装结果的字段完整性和信息隔离正确性
